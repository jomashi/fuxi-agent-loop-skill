# Acknowledgements

Fuxi is an original compact agent skill shaped by public ideas from the agent-workflow community.

## Loop Engineering Orange Book

The strongest conceptual influence is the Loop Engineering Orange Book by alchaincyf:

- Repository: https://github.com/alchaincyf/loop-engineering-orange-book
- License at the time this project was prepared: MIT License

Fuxi adapts the broad idea that durable agent work needs more than a prompt. It needs a loop: discovery, handoff, verification, persistence, scheduling, and stop conditions.

## Public agent-discipline discussions

Fuxi also reflects general lessons from public discussions around agent behavior, prompt robustness, tool-use discipline, and multi-agent handoff design.

This project does not include copied proprietary system prompts, leaked prompt text, private configuration, credentials, or user-specific workflow data.

## Attribution philosophy

If a source is licensed and directly reused, preserve its license and attribution.

If a source only influenced the design at the level of ideas, acknowledge the influence without copying its text.
